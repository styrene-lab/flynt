# Welcome to Codex

This is your first note. Everything you write here is saved as a plain file on your computer.

## Try these things

### Create a new note
Press **Cmd+N** or click the **+** button in the sidebar.

### Link notes together
Type `[[` and the name of another note to create a link. Links show up in the knowledge graph.

### Make a checklist
- [ ] Try creating a note
- [ ] Link two notes with [[wikilinks]]
- [ ] Open the graph view (Cmd+3)
- [ ] Create a drawing (Cmd+Shift+D)

### Keyboard shortcuts

| Shortcut | What it does |
|----------|-------------|
| Cmd+N | New note |
| Cmd+D | Today's daily note |
| Cmd+Shift+D | New drawing |
| Cmd+P | Command palette |
| Cmd+1 | Notes view |
| Cmd+2 | Board view |
| Cmd+3 | Graph view |
| Cmd+B | Bold text |
| Cmd+I | Italic text |
| Cmd+S | Save |

### Daily notes
Press **Cmd+D** to create today's note. It's a good place to plan your day and jot things down.

### Drawings
Press **Cmd+Shift+D** to open a drawing canvas. Draw diagrams, sketches, or mind maps.

## Your files

Your notes live in a folder on your computer. You own them — no cloud account needed. If you want to sync between devices, you can set up Git sync later in Settings.
